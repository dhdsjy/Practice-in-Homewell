### Kmeans算法回顾

**算法思想** 通过各种距离（欧式、余弦）计算两个点之前的距离，将距离最近的点归为一类

**缺点** 只能聚成球形簇

**计算步骤**

+ 随机选择k个聚类中心，分别计算其他点到k个聚类中心间的距离
+ 分别将到聚类中心点最近的一个点归类为相应的簇中并计算更新后的聚类中心
