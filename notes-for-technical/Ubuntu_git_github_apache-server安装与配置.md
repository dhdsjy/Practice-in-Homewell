### git 原理以及准备工作

git一般可以分为三个区：工作区（即本地电脑所见区）、缓存区、远程库。所以使用git和同事进行协同办公时，首先需要讲本地电脑和远程库（服务器、github）进行关联（关联指SSH的验证）。可以参考简书上的一篇博客，写的比较全 [git 命令学习](http://www.jianshu.com/p/7e2b0de68ed6)。
